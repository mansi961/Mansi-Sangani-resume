Software Development Projects

Language & Tools Used

I developed a parser for a custom Mini Language Compiler (MLC) using C++, applying core concepts of compiler design such as lexical analysis and syntax parsing. The project utilized tools like Lex/Yacc and Flex/Bison to build a tokenizer and parser capable of analyzing source code written in MLC. It was developed in environments such as Visual Studio, Code::Blocks, and G++ through the terminal. This project enhanced my understanding of language processing, grammar validation, and the inner workings of compilers, while also strengthening my C++ programming and debugging skills.
